# Course Project — Учет задач и ресурсов

Учебный JavaFX-проект для моделирования учета задач, ресурсов и сроков выполнения проекта
с учетом зависимостей между задачами и занятости сотрудников.

Приложение позволяет:
- создавать задачи с трудоемкостью и дедлайнами;
- назначать сотрудников на задачи;
- задавать зависимости между задачами;
- автоматически рассчитывать дату завершения проекта.

---

## Основные возможности

-  Таблица задач с параметрами:
    - название;
    - трудоемкость (часы);
    - статус;
    - дедлайн;
    - исполнители;
    - зависимости.
-  Список сотрудников проекта.
-  Добавление и удаление задач.
-  Поддержка зависимостей между задачами.
-  Автоматический расчет даты завершения проекта
  с учетом:
    - зависимостей;
    - доступности сотрудников;
    - длительности задач (8 часов = 1 рабочий день).

---

## Используемые технологии

- Java 17+

---

## Логика планирования

Расчет даты завершения проекта выполняется в классе `ProjectScheduler`:

1. Все сотрудники сбрасываются в состояние «свободен с сегодняшнего дня».
2. Задачи сортируются топологически по зависимостям.
3. Для каждой задачи определяется дата старта:
    - не раньше завершения всех зависимостей;
    - не раньше освобождения всех назначенных сотрудников.
4. Вычисляется дата завершения задачи.
5. Сотрудники помечаются как занятые до даты завершения задачи.
6. Итоговая дата проекта — максимальная дата завершения всех задач.
